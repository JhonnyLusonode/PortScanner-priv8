# PortScanner-priv8
Simple PortScanner

How to use:<br>
- Usage: ./scanport.py <IP address> <start port> <end port>
- Example: ./scanport.py 192.168.1.10 1 10000 <br>

- Usage: ./scanport.py <network> <start port> <end port> -n
- Example: ./scanport.py 192.168.1 1 65535 -n <br><br>
